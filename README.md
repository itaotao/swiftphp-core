# SwiftPHP Core

高性能 PHP 框架，结合了 ThinkPHP 的易用性和 Workerman 的高性能。

## 安装

```bash
composer require swiftphp/core
```

## 核心组件

- **Auth** - JWT 身份验证
- **Cache** - 缓存管理
- **Container** - 服务容器/依赖注入
- **Database** - 数据库连接池和查询构建器
- **Model** - ORM 模型，支持关联关系
- **Redis** - Redis 客户端
- **Request/Response** - HTTP 请求/响应处理
- **Routing** - 路由系统
- **Session/Cookie** - 会话和 Cookie 管理
- **Validate** - 数据验证
- **Log** - 日志系统
- **I18n** - 国际化支持
- **Middleware** - 中间件支持
